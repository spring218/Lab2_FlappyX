Flappy X

A personalized Flappy Bird game with AI integration

About the Game

This is a personalized version of the classic Flappy Bird game, featuring:

  • Custom character using your profile picture
  • AI integration using NEAT (NeuroEvolution of Augmenting Topologies)
  • Customizable game settings
  • Score tracking and high scores

Game Screenshot

How to Play

  • Press SPACE to make the bird flap
  • Press R to restart after game over
  • Press A to toggle AI control

Technical Details

This game is built using:

  • Python with Pygame
  • NEAT Python for AI implementation
  • Pillow for image processing
